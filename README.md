# Umbra.nvim

A celestial Neovim colorscheme with carefully crafted colors.

## Installation

Using lazy.nvim:

```lua
{
  "marcusschimizzi/umbra.nvim",
  lazy = false,
  priority = 1000,
  opts = {
    -- your configuration
  },
}
```
